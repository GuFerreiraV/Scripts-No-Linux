
## Scripts
- Por enquanto só criei 2 scripts simples. 
- O primeiro (clean_downloads) limpa arquivos que estão a mais de 7 dias no diretório Downloads
- O segundo (inventario_hardware) exibe informações do sistema, até informações da BIOS

## Política de permissão
- Abra o PowerShell como admnistrador
- Rode o comando abaixo para permitir a execução de scripts shell
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```
- Pressione 'Y' para confirmar tudo.
